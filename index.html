Based on this site randomstringtocsscolor.com

I decided to use a different way to calculate the colors. I use CRC-32 (Cyclic Redundancy Check) to convert the text into 32 bits. Using this http://www.simplycalc.com/crc32-source.php as the algorithm.

How does it work?

  • Take the text and run it through the CRC algorithm
  • Split result into 4 numbers (1 byte each)
  • The first 3 bytes are used as the RGB colors. Each byte ranges from 0 - 255. The last byte is used to calculate a percentage of 255. This percentage is the alpha in the color.
