Text to CSS color

random string to css color

2015 by Christian Fredrik Linchausen

Based on this site randomstringtocsscolor.com

It uses CRC-32 to convert the text into 32 bits. Using this simplycalc.com as the algorthm.

How does this work?

  • Compute the CRC from the text
  • Split result into 4 numbers (1 byte each)
  • The first 3 bytes are used as the RGB, and the last is used as a pecentage of 255.
